# 常用正则表达式 #
**今天无聊翻看以前做的一些东西，发现大部分现在在用的东西以前都学习过，并且做了详细的笔记与讲解，温故而知新，大概是现在最需要做的，去翻翻以前的代码吧！**

**附上以前写的一些正则表达式**


		'require'   =>  '/.+/',
		'email'     =>  '/^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$/',
		'url'       =>  '/^http(s?):\/\/(?:[A-za-z0-9-]+\.)+[A-za-z]{2,4}(?:[\/\?#][\/=\?%\-&~`@[\]\':+!\.#\w]*)?$/',
		'currency'  =>  '/^\d+(\.\d+)?$/',
		'number'    =>  '/^\d+$/',
		'zip'       =>  '/^\d{6}$/',
		'integer'   =>  '/^[-\+]?\d+$/',
		'double'    =>  '/^[-\+]?\d+(\.\d+)?$/',
		'english'   =>  '/^[A-Za-z]+$/',
		'qq'		=>	'/^\d{5,11}$/',
		'mobile'	=>	'/^1(3|4|5|7|8)\d{9}$/'
