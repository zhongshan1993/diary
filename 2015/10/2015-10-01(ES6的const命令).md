# ES6的const命令 #

**一整天被负能量环绕，总想着出去玩，又荒废了一天啊！！！必须调整过来！**


`const`用来声明常量。一旦声明，其值就不可改变。

    const PI = 3.1415;
	PI// 3.1415

	PI = 3;
	PI// 3.1415

	const PI = 3.2;
	PI// 3.1415


上面的代码表明改变常量的值是不起作用的。对常量的赋值不会报错，只会默默失败。


`const`的作用域与`let`命令相同： 只在声明所在的块级作用域内有效。


	if (condition) {
		const MAX = 12;
	}
	
	// MAX常量在此处不可得


`const`声明的常量，也与`let`一样不可重复声明。
